Anomaly Detection
in Healthcare

A Two-Stage Hybrid Anomaly Detector (TS-HAD) that combines deep learning with unsupervised scoring to flag critical deviations in Electronic Health Records in real time.

  • Designed a Deep Autoencoder + Isolation Forest pipeline for high-dimensional EHR feature learning and outlier scoring.
  • Minimized False Positive Rate by mapping patient observations to a compact latent space with density-independent scoring.
  • Built a microservices architecture with Kafka data ingestion, a preprocessing module, TS-HAD model service, and an alerting dashboard integrated with clinical decision support systems.
  • Evaluated using AUC-ROC and F1-score, outperforming OC-SVM, standalone Isolation Forest, and threshold-based rule systems.
